# cssBasics
###### A simple course about css used to teach the basics of positioning at HackLab ITM -conocimiento colaborativo-

## Instructions

Clone this repository

```bash
$ git clone https://github.com/Jberivera/cssBasics.git
```

Now, you can start to navigate to different states of the repository

> Note that you can only use one option at time from the bottom array, for example: ``$ git checkout c1``

```bash
$ git checkout [c1, c2, c3, ... , cn]
```
